# FPGA Implementation

## Tools

- Xilinx Vivado
- SystemVerilog
- FPGA Board (Optional)

## Steps

1. Create Vivado Project
2. Add RTL Files
3. Add Constraints File
4. Run Synthesis
5. Run Implementation
6. Generate Bitstream
7. Program FPGA

## LED Mapping

| LED | Signal |
|------|------|
| LED0 | NS_R |
| LED1 | NS_Y |
| LED2 | NS_G |
| LED3 | EW_R |
| LED4 | EW_Y |
| LED5 | EW_G |
| LED6 | PED_WALK |
